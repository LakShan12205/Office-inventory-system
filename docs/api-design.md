# API Design

## Auth

- `POST /api/auth/register`
- `POST /api/auth/login`
- `GET /api/auth/me`

## Dashboard

- `GET /api/dashboard/summary`

## Stores

- `GET /api/stores`
- `POST /api/stores`
- `PATCH /api/stores/:id`

## Monitored employees

- `GET /api/employees`
- `POST /api/employees`
- `PATCH /api/employees/:id`

## Next-phase modules

- `GET /api/schedules`
- `POST /api/schedules`
- `POST /api/observations`
- `GET /api/violations`
- `GET /api/reports/weekly`
